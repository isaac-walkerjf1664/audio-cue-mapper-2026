## Overview

Audio Cue Mapper is a browser-first analysis utility for building sound cues around motion design and synchronization tasks. It emphasizes fast visual review, cue placement, and timeline reading, letting you turn media into organized timing data without bringing in a separate desktop environment.

The tool is aimed at production pipelines where audio analysis needs to feed directly into animation software. Because it processes locally in the client and can export After Effects JSX, it suits both individuals and teams that want a compact way to inspect waveforms, detect events, and move toward delivery sooner.

## What it offers

- Browser-based audio and video file analysis
- Waveform display for visual cue review
- Spectral flux onset detection for event spotting
- Peak picking and tiering for organizing cue candidates
- Structural segmentation for section-level analysis
- Tempo estimation to support timing decisions
- After Effects JSX export for motion design workflows
- Client-side processing with no dependencies

## How to use it

1. Load an audio or video file into the interface.
2. Review the waveform and identify timing landmarks.
3. Run onset detection to surface likely cue points.
4. Use peak picking, tiering, and section analysis to refine the structure.
5. Check tempo estimation when syncing to motion or animation timing.
6. Export the results as After Effects JSX for use in your workflow.

Typical workflow:

- import media
- inspect waveform
- detect onsets
- organize cues
- export to After Effects
